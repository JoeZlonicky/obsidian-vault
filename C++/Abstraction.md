### Description
Abstraction is hiding the details behind a concept, resulting in an easy-to-use interface.
### Examples
```run-cpp
class Car {
	void drive() {
		// Code...
	}
};

int main() {
	Car myCar;
	myCar.drive();  // I don't have to care about the details
	return 0;
}
```
